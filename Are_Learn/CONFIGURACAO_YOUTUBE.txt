┌─────────────────────────────────────────────────────────────────────┐
│  🐛 SOLUÇÃO DE PROBLEMAS                                            │
└─────────────────────────────────────────────────────────────────────┘

❌ Erro "API Key inválida"
   → Verifique .env.local
   → Reinicie o servidor (npm run dev)

❌ Player não aparece
   → Aguarde 2-3 segundos (API do YouTube)
   → Verifique console do navegador (F12)

❌ Vídeo não carrega
   → Confirme que o vídeo é público
   → Teste com outro vídeo

❌ Busca não funciona
   → Verifique a API Key
   → Confirme que tem internet
   → Veja console (F12) para erros

📖 Mais soluções: TEST_YOUTUBE.md
